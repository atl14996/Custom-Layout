THIS IS AN APP THAT SHOWS CUSTOM ANIMATIONS FOR TRANSITIONS BETWEEN ACTIVITIES.  WHEN THE USER CLICKS A NEW ACTIVITY, AN ANIMATION WILL BE DISPLAYED FOLLOWING THE ORIGINAL ACTIVITY AND A DIFFERENT ANIMATION IS DISPLAYED FOLLOWING THE SECOND ACTIVITY.  THIS APPLICATION USES A CUSTOM LAYOUT (VIEWGROUP) AS A LINEAR LAYOUT PROGRAMMED IN JAVA EMBEDDED IN A NORMAL XML LAYOUT.
